# Steganografia
Queso progetto scolastico prevede di creare un programma che "nasconda" un messaggio in un'immagine in formato PGM senza alternarne l'aspetto visivo.

## Esegui il programma
Per eseguire il programma scrivi:
```bash
make quickrun
```
